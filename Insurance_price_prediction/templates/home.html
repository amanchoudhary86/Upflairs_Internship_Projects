Project Overview

This project is designed to forecast insurance premium charges based on various demographic and health-related factors. The approach combines data analysis with machine learning techniques to provide accurate predictions.

Data Preparation: The dataset includes crucial features such as age, gender, BMI, number of children, smoking status, and region. Data preprocessing techniques like one-hot encoding and converting categorical variables to numerical formats were employed to prepare the data for analysis.

Feature Engineering: BMI was categorized into different health classes (Underweight, Healthy Weight, Overweight, Obese) to better understand its impact on insurance charges.

Model Training: The following models were trained and evaluated for prediction accuracy:

  • Linear Regression: Used to model the linear relationships between the features and the insurance charges.
  • Decision Tree Regressor: Implemented to capture non-linear relationships and interactions between features.
  • Random Forest Regressor: Employed to enhance prediction accuracy through ensemble learning techniques.

Model Evaluation: Each model was evaluated based on training and testing scores to determine their effectiveness. The best-performing models were retained for future use.

Data Visualization: Various visualization tools were utilized, including heatmaps and bar charts, to demonstrate the relationships between health categories and model performance.

Outcome: The project effectively predicts insurance premiums, providing valuable insights for both customers and insurers, helping in financial planning and decision-making.
